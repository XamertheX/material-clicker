# Date Utility
## Functions
- `formatDate(date)`:
  - Formats the date into a human readable string.
  - `date`: Date, The date to format.

## Usage

## Usage
1. Import the utility.

```js
import { formatDate } from 'util/date';
```

2. Use It

```js
// Example


console.log(formatDate(new Date));
// Expected output: "2019-02-15 at 8:32pm"
```
